# Balanced Brackets Checker

## Описание

Этот проект проверяет, сбалансирована ли строка со скобками. Поддерживаются скобки трёх типов: (), [], {}.

## Структура

- `stack.py` — реализация структуры данных стек.
- `check_brackets.py` — функция проверки сбалансированности.
- `examples.py` — примеры использования.
- `.gitignore` — исключения для Git.

## Как запустить

```bash
python examples.py
